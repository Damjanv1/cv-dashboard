# Damjan Vlastelica — CV Dashboard

An interactive, editorial-style CV dashboard for Damjan Vlastelica, Data & Analytics Leader.

- Single-file HTML with light / dark theme toggle
- Chart.js visualisations: Gantt career timeline, stacked focus-over-time, industry radar, donut
- "Analytics POV" section showcasing 5 favourite chart types + a data-storytelling walkthrough
- Brand wordmarks for past clients and engagements

## Run locally

Open `Damjan-Vlastelica-Dashboard-CV.html` in any modern browser.

## Publish

Enable GitHub Pages on this repo (Settings → Pages → Deploy from branch → `main` / root), then visit:

`https://<your-username>.github.io/cv-dashboard/Damjan-Vlastelica-Dashboard-CV.html`
